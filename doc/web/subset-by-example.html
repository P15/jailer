Subset By Example is an easy way to create extraction models.

Use the Data Browser to collect all the rows to be extracted and let Jailer create a model for that subset.

Objective

Exporting all the data related to the employee named 'Scott' into a SQL-script.

Step 1. Setup the database

Do the steps 1 and 2 from the tutorial "SQL Export", if you don't already have done it.
